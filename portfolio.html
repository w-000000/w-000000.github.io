🙋 자기소개

안녕하세요. 전북대학교 소프트웨어공학과에서 소프트웨어와 인공지능을 공부하고 있는 임창우입니다.

소프트웨어 결함 예측, 시스템 자원 이상 탐지, 멀티에이전트 기반 면접 분석 시스템, 스마트 농업 자동화 시스템 등 다양한 프로젝트를 통해 AI 모델뿐만 아니라 실제 데이터 처리 구조와 시스템 구현 경험을 쌓았습니다.

단순히 기능을 구현하는 것에서 끝나는 것이 아니라, 실제 환경에서 안정적으로 동작하고 사용자의 문제를 해결하는 AI 서비스를 만드는 개발자를 목표로 하고 있습니다.

🎤

멀티에이전트 기반 면접 표현 습관 분석 시스템

AI · 음성 분석 · 영상 분석 · 멀티에이전트

면접자의 음성과 영상 데이터를 실시간으로 수집하고, 발화 습관과 비언어적 행동을 분석한 뒤 LLM 기반 피드백과 추가 질문을 제공하는 시스템입니다.

주요 역할

  • Whisper 기반 STT와 발화 텍스트 분석
  • librosa를 활용한 발화 속도, 무음 구간 및 MFCC 분석
  • OpenCV와 MediaPipe를 활용한 얼굴, 손, 상체 랜드마크 분석
  • Speech, Gesture, Analysis, Feedback Agent 구조 설계
  • Qwen API 기반 면접 답변 평가와 추가 질문 생성 기능 구현

문제 해결 경험

음성과 영상의 처리 속도가 달라 발생하는 멀티모달 동기화 문제를 timestamp, queue, buffer 기반 구조로 해결했습니다. 프레임 누락과 랜드마크 미검출 문제에는 결측 프레임 보간과 window 단위 예외 처리를 적용했습니다.

Python Whisper MediaPipe OpenCV librosa Qwen API
